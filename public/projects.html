Automatic Attendance Marker

IoT & Edge Computing Project

  • ESP32
  • RFID
  • C++
  • Google Apps Script

An IoT-based system designed to replace manual attendance processes with a fast, contactless solution. It uses an ESP32 microcontroller paired with an RFID RC522 reader to identify users and log entries in real-time.

  • Designed firmware to handle RFID detection, mode control, and network communication.
  • Transmitted attendance data over Wi-Fi via HTTP to a backend layer powered by Google Apps Script.
  • Integrated real-time data sync directly into Google Sheets for instant administrative access.
  • Implemented physical toggles for registration vs. attendance modes with LED feedback.
